Schema Overview

Students ( id, name, gender, age )
Courses ( id, name, instructor )
Enrollments ( id, studentid → Students.id, courseid → Courses.id, grade, marks )
Note: marks (integer, 0–100) is added to Enrollments to support SUM / AVG / MIN / MAX examples on a meaningful numeric field.
Note: Student S106 Frank has no enrollment (demonstrates LEFT JOIN NULL row). Course C104 Biology has no students (demonstrates RIGHT JOIN NULL row).

PART 1 — SQL

1.1 Create Tables

Students
CREATE TABLE Students (
    id      VARCHAR(10)  PRIMARY KEY,
    name    VARCHAR(100) NOT NULL,
    gender  VARCHAR(10),
    age     INT
);
Courses
CREATE TABLE Courses (
    id         VARCHAR(10)  PRIMARY KEY,
    name       VARCHAR(100) NOT NULL,
    instructor VARCHAR(100)
);
Enrollments
CREATE TABLE Enrollments (
    id        VARCHAR(10) PRIMARY KEY,
    studentid VARCHAR(10) REFERENCES Students(id),
    courseid  VARCHAR(10) REFERENCES Courses(id),
    grade     VARCHAR(5),
    marks     INT
);

HAVING filters groups after GROUP BY. WHERE cannot be used with aggregate functions — use HAVING instead.
Courses with more than 2 enrollments

FROM     Enrollments
GROUP BY courseid

1.6 JOINs

INNER JOIN returns only matched rows. LEFT JOIN keeps all rows from the left table. RIGHT JOIN keeps all rows from the right table.
INNER JOIN — Students with their enrolled courses and marks

       c.name    AS course_name,
       e.grade,
       e.marks
FROM   Students    s
JOIN   Enrollments e ON s.id = e.studentid
JOIN   Courses     c ON c.id = e.courseid
ORDER BY s.name;

2.5 $lookup — JOIN equivalent

INNER JOIN equivalent — Students with enrolled course names and marks
db.enrollments.aggregate([
    {
        $lookup: {
            from:         "students",
            localField:   "studentid",
            foreignField: "id",
            as:           "student"
        }
    },
    { $unwind: "$student" },
    {
        $lookup: {
            from:         "courses",
            localField:   "courseid",
            foreignField: "id",
            as:           "course"
        }
    },
    { $unwind: "$course" },
    {
        $project: {
            _id:          0,
            student_name: "$student.name",
            course_name:  "$course.name",
            grade:        1,
            marks:        1
        }
    },
    { $sort: { student_name: 1 } }
]);

Subquery-like — Students who scored above the overall average (80.5)
// Step 1: Calculate the overall average
const avgResult = db.enrollments.aggregate([
    { $group: { _id: null, avg_marks: { $avg: "$marks" } } }
]).toArray();

const avgMarks = avgResult[0].avg_marks; // 80.5

// Step 2: Find enrollments above average, then join student names
db.enrollments.aggregate([
    { $match: { marks: { $gt: avgMarks } } },
    {
        $lookup: {
            from:         "students",
            localField:   "studentid",
            foreignField: "id",
            as:           "student"
        }
    },
    { $unwind: "$student" },
    {
        $project: {
            _id:          0,
            student_name: "$student.name",
            marks:        1
        }
    },
    { $sort: { marks: -1 } }
]);
Result:
{ "student_name": "Carol", "marks": 91 }
{ "student_name": "Carol", "marks": 90 }
{ "student_name": "Alice", "marks": 88 }
{ "student_name": "Alice", "marks": 85 }
